# 🧠 Smart Notes App

A simple and responsive **Notes Web Application** built using HTML, CSS, and JavaScript.  
This project allows users to create, view, and delete notes easily in the browser.

---

## 🌐 Live Demo

👉 https://easha-sharma.github.io/SmartNotes/

---

## 🚀 Features

- 📝 Add new notes  
- 🗑️ Delete notes  
- ⚡ Fast and lightweight  
- 📱 Responsive design  
- 🎨 Simple UI using HTML/CSS/JS  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript  
- GitHub Pages (Hosting)  
- Docker (Containerization)

---

## 📁 Project Structure

- index.html  
- style.css  
- script.js
